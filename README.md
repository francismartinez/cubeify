## Install
`yarn global add cubeify`

## Usage

```
$ cubeify textwoo
$
        T E X T W O O !
      / E           / O
    /   X         /   O
  /     T       /     W
T E X T W O O !       T
E       O     O       X

T       ! O O W T X E T
W     /       O     /
O   /         O   /

! O O W T X E T

$
```
